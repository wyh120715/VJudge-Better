# VJudge-Better

一个好用的 Virtual Judge 美化插件

## ✨ 功能特性

- 🎨 **字体切换**: 自动检测浏览器已安装的字体并支持切换
- 🖼️ **自定义背景**: 支持上传 JPG、PNG、WebP、GIF（可选）、MP4/WebM（可选）作为背景
- 🌫️ **透明度调节**: 可自由调节背景透明度
- 📝 **提交界面美化**: 优化 VJudge 题目提交界面的视觉效果
- 🏆 **榜单界面美化**: 美化比赛榜单，包括奖牌颜色、AC/WA 状态等
- 🎯 **全局美化**: 导航栏、卡片、按钮、表格、滚动条等统一美化

## 🚀 安装方法

1. 确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 点击 [app.js](./app.js) 文件
3. Tampermonkey 会自动识别并提示安装
4. 访问 VJudge 网站即可生效

## 📖 使用说明

### 打开设置面板
- **方法一**: 点击 Tampermonkey 图标 → 找到 "⚙️ VJudgeBetter 设置"
- **方法二**: 按快捷键 `Alt + S`

### 设置项说明

| 设置项 | 说明 |
|--------|------|
| 字体选择 | 从检测到的可用字体中选择喜欢的字体 |
| 背景图片 | 上传本地图片（JPG/PNG/WebP）或视频（MP4/WebM） |
| 背景透明度 | 拖动滑块调节背景透明度（0% - 100%） |
| GIF 背景 | 启用后可使用 GIF 动图作为背景（可能影响性能） |
| 视频背景 | 启用后可使用 MP4/WebM 视频作为背景 |

## 🎨 美化效果

### 提交界面
- 毛玻璃效果表单
- 渐变紫色提交按钮
- 平滑的输入框聚焦动画

### 榜单界面
- 渐变紫色表头
- 金银铜牌渐变效果
- AC/WA 状态颜色优化
- 悬停高亮效果

### 其他美化
- 渐变紫色导航栏
- 圆角卡片设计
- 自定义渐变滚动条

## ⚠️ 注意事项

1. 视频背景功能需要手动启用，且视频会自动静音循环播放
2. GIF 背景可能影响页面性能，建议谨慎使用
3. 大尺寸图片建议先压缩后再上传
4. 所有配置数据存储在 Tampermonkey 的本地存储中

## 🛠️ 技术栈

- Vanilla JavaScript
- CSS3 (backdrop-filter, gradients, animations)
- Tampermonkey API
